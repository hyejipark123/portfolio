STARHORSE:DASHBOARD

Starhorse는 인공지능 딥러닝을 활용한 경마 순위 예측 프로그램입니다.
기상청과 마사회에서 제공하는 OPEN API 데이터를 자동으로 수집하며,
순위예측 프로그램 내 자료로 활용할 수 있도록 저장하는
대시보드 형태의 반응형 웹페이지입니다.

기여도기획 및 디자인 30%, 퍼블리싱 100%

DATE2022.09 ~ 2022.10

TYPEResponsive Web

WIREFRAME

반드시 필요로 하는 정보들을 명확하게 나뉘어 노출하고,
효율적인 UX를 제공하기 위해 설계하였습니다.

PROJECT
GOAL

기존 경마 경주 분석 및 예측정보를 제공하는 인쇄물 형태의 경마 정보지보다 보기 편리해야 된다는 점을 고려하여 스크롤 기능, 페이지 이동 버튼 등을 사용하여 사용자가 한 페이지 내에서 필요한 정보를 확인할 수 있도록 목표를 두었습니다.

MAIN

데스크탑, 태블릿, 모바일 각 레이아웃으로 반응형 홈페이지를 제작하였고, 아이콘을 사용하여 데이터를 더 돋보이도록 하였습니다. 블루 컬러 계열을 사용하여 정보를 직관적으로 보이도록 표현했습니다.

정보지 현황

다양한 경마 정보지의 OCR된 예측 정보 데이터를 확인할 수 있습니다.
원하는 정보지 클릭 시 날짜 및 회차별로 예측 정보를 등록, 수정하며 각 데이터를 관리할 수 있습니다.

승부예측

월별 경기 일정을 확인할 수 있습니다. 날짜 클릭 시 해당 날짜에 회차별 요약된 경기 데이터가 우측에 화면이 펼쳐지며, 해당 회차에 경기 결과, AI 예측 데이터, 예측 적중률, 경기 정보 등 데이터를 확인할 수 있습니다.

ANOTHER SCREENS

MOBILE

Starhores 대시보드는 사용자에 따른 반응형 웹으로 제작하였습니다. 모바일 디바이스는 최소한의 데이터 확인이 필요로 되는 레이아웃에 슬라이드 혹은 스크롤을 추가하였습니다.
