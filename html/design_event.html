옥토북스 이벤트 페이지

옥토북스에서 매 월 진행하는 고정이벤트를 디자인하였습니다.
이벤트 페이지의 경우 브랜드 컬러와 비슷한 색상을 사용하며
최대한 깔끔하고 내용이 잘 전달될 수 있도록 작업하였습니다.

케이원코스메틱 이벤트 페이지

심플리카몰 자사에서 진행하는 이벤트의 경우 브랜드 컬러에 맞춰 제작하였으며
타 쇼핑 플랫폼에서 진행하는 이벤트 인트로 배너의 경우
이벤트 콘셉트에 맞춰 디자인하였습니다.

EVENT PAGE DESIGN 03

힐링안과 이벤트 페이지

힐링안과에서 진행하는 이벤트 상세페이지를 제작하였습니다.
상세페이지의 경우 이미지를 사용하여 관련한 내용에 대한 이해도를 높였으며,
광고 배너의 경우 최대한 간결하게 디자인하였습니다.

EVENT PAGE DESIGN 04

마마픽 이벤트 페이지

일러스트 이미지를 활용하여 포스터 느낌으로 디자인하였으며,
리오더의 경우 기존 이미지에서 날짜 부분만 수정하였습니다.
