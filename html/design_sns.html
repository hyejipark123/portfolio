옥토북스 SNS 콘텐츠

옥토북스는 유아˙어린이를 키우는 엄마를 주요 타켓팅으로 잡았습니다.
SNS 콘텐츠로 해외 유아 서적 정보가 업로드 되며,
추가로 자사 이벤트를 진행하여 유입 고객을 늘렸습니다.

BEFORE

일별 도서 주제를 선정하여 해당 주제에 맞는 타이틀, 이미지를
사용하여 메인 이미지를 제작하고, 도서 줄거리를 작성하여
포스팅을 제작하였습니다.

AFTER

데일리 포스트의 경우 많은 시간을 들이지 않기위해
지정 도서에 이미지 및 캐릭터를 사용하여
깔끔한 이미지를 제작하였습니다.

CONTENTS

오프라인 이벤트, 명절 인사,
도서 컨텐츠등 브랜드 컨셉에 맞춰
밝은 색상을 사용하여 디자인하였습니다.

EVENT CONTENTS

이벤트 페이지 레이아웃을 리사이징하여
SNS에 업로드 하였습니다.

@simplica_korea

케이원코스메틱 SNS 콘텐츠

브랜드 피드에는 브랜드 컬러에 벗어나지 않도록 제품과 비슷한 색감을
사용하고 제품이 돋보이도록 깔끔하고 안정적인 느낌을 주며,
SNS 광고 이미지는 내용을 강조하여 클릭율을 높이도록 디자인 하였습니다.

FEED DESIGN

케이원코스메틱의 경우 제품 합성컷을
제작하여 깔끔한 제품 이미지,
매 월 이벤트 내용으로 제작합니다.

ADVERTISEMENT DESIGN

광고 이미지의 경우 클라이언트의 요청사항 반영
혹은지난 광고에서 유입율, 클릭율이 높았던 이미지를
참고하여 제작하였습니다.
