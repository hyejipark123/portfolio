Project Overview

소마요가코리아는 소마틱스와 요가를 접목한 소마요가를 기본 바탕으로 하는 요가 스튜디오입니다.
기존 홈페이지의 불필요한 색상 사용으로 인해 가독성이 떨어지고 UI/UX의 불편한 점을 분석하였습니다.
자연 친화적인 느낌의 초록색을 포인트 컬러로 사용하여 심플하면서 이미지를 사용하여 사용성 높은 웹페이지로 리뉴얼 하였습니다.

Unorganized Text

서브 텍스트 내용의 가독성이 떨어집니다.

DESIGN CONCEPT

요가하면 떠오르는 편안함, 역동적, 자연, 건강등 단어들을 컨셉으로 지정하였습니다.
해당 브랜드 특성상 보여지는 내용이 제한적이기에 이미지화하여 두드러지게 표현하였습니다.

Project Goal

단순한 운동이 아닌 나 자신 스스로 이해하고 인지를 깨워나가는 시간

요가 외에도 재활차원 혹은 통증케어를 위한 프로그램.
브랜드가 추구하는 흔한 요가 운동이 아닌 나 자신의 시점과 타인의 시점을 통해
인지를 깨워나가는 시간을 전달 할 수 있는 UI요소를 구현함을 목표로 합니다.
